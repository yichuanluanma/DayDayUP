1、目录结构
/
/bin 		存放二进制可执行文件
/sbin		（super user binary） 超级用户root 才能访问，存放二进制可执行文件
/etc		存放系统配置文件
/usr 		（Unix shared resources）存放共享的系统资源
/home		存放用户文件的根目录
/root		超级用户目录
/dev		存放设备文件
/lib		存放跟文件系统中的程序运行所需要的共享库 和内核模块
/mnt		系统管理员安装临时文件系统的安装点
/boot 		存放系统引导时使用的各种文件
/tmp 		临时文件
/var 		存放运行时需要改变数据的文件
